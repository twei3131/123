//课程介绍页面，获取数据
1、获取课程信息
SELECT curnum AS 选课人数,singalIntro AS 课程简介,intro AS 课程介绍
	FROM `subject`
		WHERE subId = #{subId}
返回subject对象
2、获取教师信息
SELECT a.teaId AS 教师编号,a.nickname AS 昵称
	FROM teacher a JOIN subtotea b ON a.teaId = b.teaId
		WHERE b.subId = #{subId}
返回teacher对象
3、获取教师组合粉丝数量
SELECT COUNT(b.stuId) AS 数量
	FROM subtotea a JOIN stutotea b ON a.teaId = b.teaId
		WHERE a.subId = #{subId} AND b.isFans = 1
返回一个int类型变量
4、获取用户是否喜爱、关注和点赞的结果集
SELECT isLove AS 喜爱状况,isPraise AS 点赞状态,isFocus AS 关注状态
	FROM stutosub
		WHERE stuId = #{stuId} AND subId = #{subId}
返回一个state对象
